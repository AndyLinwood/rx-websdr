# WebSDR Server - Многодиапазонный сервер для RX888

## Описание

C++ сервер для WebSDR, использующий приемник RX888 и программу radiod. Сервер получает IQ данные через FIFO файлы и предоставляет веб-интерфейс для пользователей.

## Возможности

- **Многодиапазонность**: Поддержка неограниченного количества диапазонов
- **Гибкая настройка частот**: Каждый диапазон настраивается отдельно
- **Широкий диапазон sample rate**: От 192 кГц до 1600 кГц
- **FIFO интерфейс**: Получение IQ данных от radiod через именованные каналы
- **Веб-интерфейс**: Интеграция с существующим интерфейсом из web/pub2
- **Многопоточность**: Отдельный поток для каждого диапазона и клиента

## Требования

- Компилятор C++ с поддержкой C++17 (g++ 7+)
- Библиотека FFTW3 (`libfftw3f-dev`)
- pthread
- Linux OS

## Конфигурация

Файл конфигурации: `cfg/websdr.cfg`

### Основные параметры

- `tcpport` - порт для прослушивания (по умолчанию 80)
- `maxusers` - максимальное количество пользователей
- `initial` - начальная частота и режим работы

### Настройка диапазонов

Для каждого диапазона указываются:

```
band <название>
device <путь_к_FIFO>
samplerate <частота_дискретизации>
centerfreq <центральная_частота_кГц>
gain <усиление_дБ>
swapiq # опционально, поменять I и Q местами
extrazoom <уровень>
noiseblanker <уровень>
antenna <описание_антенны>
```

### Пример настройки диапазона

```
band 20m
device /home/websdr/fifo/fifo20mL
samplerate 384000
centerfreq 14175
gain -28
swapiq
noiseblanker 30
antenna Inverted Vee -> RX888 MkII
```

## Поддерживаемые sample rate

Сервер поддерживает следующие частоты дискретизации:
- 192000 (192 кГц)
- 384000 (384 кГц)
- 768000 (768 кГц)
- 1536000 (1536 кГц)
- 1600000 (1600 кГц)

## Интеграция с radiod

1. Настройте radiod для вывода IQ данных в FIFO файлы
2. Укажите пути к FIFO в конфигурационном файле
3. Запустите radiod для каждого диапазона
4. Запустите WebSDR сервер

### Пример запуска radiod

```bash
# Для диапазона 20m
radiod --device rx888 --fifo /home/websdr/fifo/fifo20mL --samplerate 384000 --center-freq 14175000
```

## API

### Статус сервера

```
GET /status
```

Возвращает JSON со списком диапазонов и количеством пользователей.

### Waterfall данные

```
GET /getwaterfall?band=20m&frequency=14100
```

Возвращает бинарные данные waterfall.

### Аудио поток

```
GET /getaudio?band=20m&frequency=14100&mode=lsb
```

Возвращает демодулированный аудио поток.

## Управление

- **Ctrl+C** - Остановка сервера
- **SIGHUP** - Перезагрузка конфигурации (планируется)

## Логи

Сервер выводит логи в консоль. Для фонового запуска используйте:

```bash
nohup ./bin/websdr_server cfg/websdr.cfg > websdr.log 2>&1 &
```
